Backup All Your Wordpress Installation with some easy steps with Website Backup Plugin. Backup your Database and all Wordpress files with just one plugin.

== Description ==
The Website Backup plugin allows you to create the backup for your wordpress websites. By creating the backups, you can prevent the data loss due to server crashes, malware attacks, security flaws or any other updates. With this plugin, You can create backup in two ways: File Backup and Database Backup.

* File Backup allows you to create backup of your all wordpress files like plugins, themes and other media files etc. It will create a complete backup for your whole wordpress folder in zip format. You can simply extract it and place it on your local/online directory.
* Database Backup allows you to create backup of your database tables in sql.gz format. Just extract it and upload it to your wordpress database.

## Main Feature of plugin:
* One Click Files Backup.
* One Click Database Backup.
* Keep Multiple Backups To Download your website from old stable versions.
* One Click Backups Download

== Installation ==
Upload the plugin to your Wordpress Website, Activate it.

You Are Done.

== Frequently Asked Questions ==
1) Is This Plugin Support Database And Files Backup both?
Yes, It Support Database As Well As Files Backups Both Creation and Downloading.

2) Can i Download The backups?
Yes, You Can Easly Download Backup.

3) Which file type is created for database backup?
It uses .gz (gzip) file type to create backups. which is highly compressed file type.

3) Which file type is created for Files backup?
It uses .zip (zip) file type to create backups.
